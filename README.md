# Recent Projects
## Twilio Assistant
### Twilio Alpha AI Assistant with MCP
#### #node.js, openai, mcp, promptengineering, emailapi, twilio
This project focused on creating an assistant for voice and chat channels.

I created a few tools to achieve the following:
- Get channel(voice/chat) and callSid(voice)
- Redirect the call to my mobile number(voice)
- Send email with call/chat notes

I also added an AI optimized resume to the knowledge base.

Because this AI Assistant tool is part of Twilio Alpha and doesn't appear to be moving forward, it is just an experiment.

There were some obvious limitations to it, such as accessing the callSid on voice calls. 

It also tended to ask redundant questions at times or use tools more than once redundantly.

However, the main goal was achieved.

Check it out at [/TJVaughn/twilio-personal-assistant](https://github.com/TJVaughn/twilio-personal-assistant)

Status: **Complete**

## Twilio Conversation Relay
### WebSocket Server with Golang
#### #go, openai, promptengineering, websocket, twilio
This project focused on integrating with conversation relay in a Twilio Studio flow

Once a user enters a voice call, the conversation relay is triggered within the studio flow.

It connects to a web socket server for low latency operations.

From there, the websocket server receives and returns JSON data as the user interacts with the server.

The server integrates with Open AI text completion API to answer user queries.

Status: **In Progress**
